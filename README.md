## Project Overview

This project investigates how sleep metrics, lifestyle habits, and environmental conditions predict two personal outcomes: morning feel rating (how rested I feel upon waking, 0–10) and daily productivity rating (0–10). Data was collected over 44 nights (April 1 – May 14, 2026) at Sabanci University, Tuzla, Istanbul. A dual-target model jointly predicts both outcomes and derives a personalized optimal daily routine recommendation.

## Data Sources

| Source | Features | Method |
|---|---|---|
| Apple Watch (Health XML) | Sleep stages, HRV, SpO₂, HR, steps, calories, exercise, Apple sleep score | Health app export |
| Daily Self-Log | Morning feel (0–10), productivity (0–10), caffeine, alcohol, meal timing | Manual daily log |

## Key Results

- **Morning Feel (Lasso, LOOCV):** MAE = 1.600, R² = 0.494 — all models beat the naive baseline
- **Productivity (XGBoost, LOOCV):** MAE = 2.311, R² = 0.284 — best overall for this target
- **Optimal routine:** sleep ~6.9h, last caffeine by 16:30, last meal by 19:00, ≥21,000 steps/day

---

## Known Limitations

- **Sample size (n=37):** Personal dataset; findings are not generalizable
- **Self-reporting bias:** Subjective targets (morning feel, productivity)
- **Environmental confound:** Roommate snoring elevates awake time independently of lifestyle factors
- **Weather data:** WMO 30-year climate normals used (API access unavailable in project environment)
- **Missing Watch nights (n=7):** Excluded from ML training
